3. 功能概览（当前设计基线）

ReadyKids 围绕三个核心页面设计，帮助家庭「少催促、少遗漏、少迟到、少崩溃」：

  1. 家长首页（Home）：查看孩子当日进度，点击全应用主操作 「START MORNING / 开始晨间」；底部可进入编辑流程、出门检查、周报（Plus）；
  2. 儿童流程页（Kid Flow）：一次只显示一个任务，大按钮「完成」；视觉时间（太阳/火箭进度，非焦虑红色倒计时）；完成后鼓励动画与语音；
  3. 出门前检查（Leave Home Check）：午餐盒、水瓶、作业、Chromebook、运动包、外套等清单，完成后「准备出发」。

当前版本明确不提供：AI 聊天、社交网络、成长游戏化体系、多账户家庭权限后台、Google Calendar 集成、复杂云端统计后台、医疗诊断或紧急响应。免费版与付费版均无第三方广告。

5. 常见问题

5.1 如何开始一次晨间流程？

  1. 在家长首页选择孩子（免费版仅第一个孩子可用；第二个孩子需 Plus）；
  2. 点击 「START MORNING / 开始晨间」；
  3. 进入儿童全屏流程页，按提示完成每一步并点「完成」；
  4. 全部完成后会引导至「出门前检查」。

5.2 语音不播放或没有声音

  1. 在「设置」中确认 语音提醒 已开启；
  2. 关闭 iPhone 静音拨片 / 调高媒体音量；
  3. 确认未连接仅支持通话的蓝牙设备；
  4. 语音由系统 TTS 在本机播放，不上传录音；若系统语言与 App 语言不一致，音色可能不同；
  5. Plus 用户可在设置中选择高级语音包（若已提供）。

5.3 什么是 ADHD 模式 / ADHD 高级模式？

  • 基础 ADHD 友好设计（免费）：儿童页一次只显示一个任务、少文字、无红色警告式 UI；
  • ADHD 高级模式（Plus）：隐藏步骤编号与「下一步」预告、更大完成按钮、纯视觉时间（隐藏数字时钟）等。

二者均为界面与节奏辅助，不构成医疗或心理治疗。详见 EULA。

5.4 第二个孩子显示锁定（🔒 Plus）

免费版仅支持 1 个孩子。如需为多名儿童分别记录流程，请订阅 ReadyKids Plus 或购买 Lifetime，并在应用内 Paywall 完成购买后重试。

5.5 睡前 / 放学流程在哪里？

在 编辑流程（Edit Routine） 顶部切换「早晨 / 睡前 / 放学」。免费版仅「早晨」可用；Plus 解锁其余类型。

5.6 数据会丢吗？如何备份？

数据默认保存在本机。卸载 App、清除数据或恢复设备且未备份，可能导致数据丢失。运营者无法远程恢复。

  • 在「设置」中使用 导出我的数据 保存 JSON 副本（建议定期备份到 iCloud Drive / 电脑）；
  • 使用 删除所有数据 可清除本地记录（不可撤销）；
  • 启用 iCloud 整机备份时，应用数据可能包含在备份中（由 Apple 管理，非运营者可访问）。

5.7 周报是空的或无法打开

周报需先完成若干次晨间流程后才会生成统计。周报为 Plus 功能；免费用户点击会引导至订阅页。

5.8 出门检查清单可以单独用吗？

可以。在家长首页底部点击 Backpack Check / 出门检查，无需先完成晨间流程也可勾选物品。

5.9 深色模式 / 语言

在应用内顶部按钮切换 🌙 深色模式；「中文 / EN」切换界面语言。法律页面语言以各 HTML 页面为准。

5.10 欢迎页想再看一次

卸载并重装会重新显示引导；或清除应用数据。运营者不提供远程重置服务。

5.11 应用崩溃或无法打开

  1. 重启设备后重试；
  2. 在 App Store 检查 ReadyKids 是否为最新版本；
  3. 若刚更新 iOS，请等待系统稳定后重试；
  4. 仍失败请邮件联系我们，附上 iOS 版本与崩溃前操作（勿附儿童隐私截图）。

我们不保证应用在所有越狱、测试版 iOS 或极端低存储设备上完美运行。
